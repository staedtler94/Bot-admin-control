# Scripts

| Script | How to run | Purpose |
|--------|------------|---------|
| `setup-dev.mjs` | `npm run setup` (from repo root) | `npm install` in root, backend, and frontend, then print the multi-terminal dev workflow |
| `setup-dev.mjs --skip-install` | `npm run setup:help` | Print the workflow only (no installs) |
| `seed-database.mjs` | `npm run seed:database` | POST sample data from `data/*.json` to the running API |

See [runbook: Quick start](../docs/runbook/Readme.md) for the full flow.
